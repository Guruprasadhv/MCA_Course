MCA103 - Database Management Systems (DBMS)

Course Description:

This course focuses on the concepts and principles behind Database Management Systems (DBMS). It covers the relational model, SQL, normalization, transaction management, and database security.

Topics Covered:

  • Introduction to DBMS
  • Relational Model and SQL
  • Normalization
  • Transactions and Concurrency Control
  • Database Security

Learning Outcomes:

  • Understand the principles of relational databases.
  • Learn how to design and optimize databases using SQL.
  • Gain knowledge of database transaction management and security.
